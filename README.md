# Crunum
## What is Crunum?
Crunum(which stand for "Crunching number") is a library for Matrix and Vector,

It only supports Lua, for now.

Purpose? For fun
## How to install it
Clone this repository
```bash
git clone https://github.com/Vgwws/crunum.git
```
Install it
```bash
make install
```
If you want to uninstall it
```bash
make uninstall
```
